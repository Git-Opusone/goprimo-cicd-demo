# GoPrimo CI/CD Demo

Sample Node.js application for learning enterprise-style CI/CD.

## Endpoints

- `/` - Application status
- `/health` - Health check
- `/api/orders` - Sample API
- `/metrics` - Prometheus metrics

## Tech Stack

- Node.js
- Express.js
- Docker
- GitHub Actions
- GitHub Container Registry
- Grafana Cloud
- Loki
- Grafana Alloy
